What is CSS?

Cascading Style Sheets

CSS or Cascading Style Sheets describes how the presentation of a document written in a markup language such as HTML. It shows how HTML elements are to be displayed on screen, paper, or in other media based on w3schools. CSS has been one of the important technology of the World Wide Web, together with HTML and JavaScript. This language was initialy released back in December 17, 1996, which is 23 years ago.

For more info go to w3schools.com
